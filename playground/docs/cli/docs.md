# docs — 文档站点

> 启动 harness-cook 文档站点，浏览指南、教程、Demo 等文档

```bash
# 启动开发服务器
harness docs

# 自定义端口
harness docs --port 8080

# 启动后自动打开浏览器
harness docs --open

# 构建静态站点
harness docs --build
```

## 参数

| 参数 | 说明 |
|------|------|
| `--port / -p` | 开发服务器端口（默认 5173） |
| `--open` | 启动后自动打开浏览器 |
| `--build` | 构建静态站点（不启动 dev server） |
